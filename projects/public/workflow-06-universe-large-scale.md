### 程序化分布算法

**银河系螺旋臂:**
```typescript
// 4条螺旋臂, 每条1250点(5000/4)
for (const arm of [0, 1, 2, 3]) {
  const armAngle = (arm / 4) * Math.PI * 2;
  for (let i = 0; i < 1250; i++) {
    const t = i / 1250;
    const r = t * 18;  // 半径0→18
    const angle = armAngle + t * 3.5;  // 螺旋角
    // 添加噪声: (Math.random()-0.5) * 2
  }
}
```

**星系沿宇宙纤维分布:**

**distTo3D()距离转换:**
```typescript
function distTo3D(d: number): number {
  // d单位: 万光年
  // 近距(0-100万光年): 对数缩放
  // 中距(100-5000万光年): 线性+压缩
  // 远距(5000万+): 进一步压缩
  if (d < 10) return d * 3;
  if (d < 100) return 30 + (d - 10) * 0.8;
  if (d < 1000) return 102 + (d - 100) * 0.15;
  return 237 + (d - 1000) * 0.03;
}
```
