# SkyGIS 工作流05 - 不可见现象3D动态渲染

## 背景
用户反馈"不是动态的，物质不是真实存在在这个系统里"

## 问题(截图分析)
1. 磁场线呈尖锐锯齿状 — Line几何体段数不够
2. 太阳风几乎不可见 — 粒子size 0.15, opacity 0.4
3. 辐射带完全看不到 — TorusGeometry opacity 0.08-0.14
4. 场景整体静态感 — 动画幅度过小(呼吸±3%, 脉动±6%)
5. 暗物质只有静态球壳 — opacity 0.025, 完全不可见

## 方案：静态几何体→动态发光粒子流

### 第一版实现(静态几何体) — 已废弃

| 现象 | 渲染方式 | 问题 |
|------|----------|------|
| 磁场 | 8条Line几何体+CatmullRomCurve3 | 锯齿,无发光,呼吸仅±3% |
| 太阳风 | 3000粒子,size 0.15,opacity 0.4 | 在暗背景上几乎不可见 |
| 辐射带 | TorusMesh,opacity 0.08-0.14 | 完全不可见 |
| 暗物质 | SphereGeometry,opacity 0.025 | 完全不可见 |
| CME | 200粒子/束,opacity 0.2 | 看不出效果 |
| 宇宙射线 | 8条线,flash阈值0.92 | 几乎不亮 |

### 第二版实现(动态粒子流) — 当前版本

#### 1. 磁场→发光粒子流
- 每个有磁场行星: 8条磁力线 × 30粒子 = 240粒子/行星
- 粒子沿偶极场路径实时流动(不是线条)
- 参数: size 0.25, opacity 0.6, 行星对应色
- 6个行星有磁场: 地球/木星/土星/天王星/海王星/水星
- 磁场倾斜角: 地球11.5°/木星9.6°/土星0°/天王星59°/海王星47°/水星0°

```typescript
// 偶极场路径: r(θ) = L·sin²(θ)
// 8条磁力线: L = 1.5, 2.5, 3.5, 4.5 各2条(对称)
// 粒子flow offset: 0→1 沿路径循环流动
```

#### 2. 太阳风→明亮径向流
- 5000粒子(之前3000)
- size 0.35(之前0.15), opacity 0.7(之前0.4)
- 黄白色(#fef08a), AdditiveBlending
- 速度0.08-0.3(之前0.05-0.15)
- 粒子从太阳径向扩散,到达maxR后重置

#### 3. 辐射带→发光粒子环
- 地球: 内环600+外环600 = 1200粒子
- 木星: 内环1000+外环1000 = 2000粒子
- 土星: 内环800+外环800 = 1600粒子
- 粒子沿环运动(不是静态mesh)
- 颜色: 青色(#22d3ee), AdditiveBlending, size 0.15, opacity 0.5

```typescript
// 辐射带粒子位置: 
// angle = baseAngle + time * speed
// x = center.x + r * cos(angle)
// y = center.y + tiltY * sin(angle)  
// z = center.z + r * sin(angle) * cos(tilt)
```

#### 4. CME→明亮喷流
- 3束, 每束500粒子(之前200)
- 方向: 0°/120°/240° (三等分)
- 粒子从太阳表面沿束方向推进
- opacity 0.3-0.6(之前0.2)

#### 5. 宇宙射线→频繁闪烁
- 12条线(之前8)
- flash阈值0.7→opacity 0.8(之前0.92→0.7)
- 更频繁可见

#### 6. 黄道光
- 2000粒子扁平分布(太阳系平面)
- 白色半透明, size 0.2, opacity 0.15

#### 7. 暗物质→流动粒子云(宇宙视图)
- 银河系大晕: 3000粒子, 半径50
- 3条暗物质纤维: 各800粒子(连接超星系团)
- 50个星系暗物质小晕: 各200粒子
- 紫色(#a855f7), AdditiveBlending, 缓慢旋转
- centerOffset各星系暗物质小晕绕中心旋转

#### 8. CMB宇宙微波背景辐射(宇宙视图)
- 程序化温度涨落纹理球壳(512×256 canvas)
- 极低opacity 0.03
- 极大半径5000

#### 9. 星际介质(宇宙视图)
- 4000粒子沿银河系4条螺旋臂
- 三色: HII红(#f87171)/分子云紫(#a855f7)/中性氢蓝(#60a5fa)

### 圆形纹理作用域修复
- 太阳系部分创建了 `sysCircleTex`(在宇宙部分circleTexture之前)
- 太阳风/辐射带/CME/黄道光使用sysCircleTex
- 宇宙部分使用circleTexture

### 交互控制
- ⚡场 按钮切换showInvisiblePhenomena
- useEffect联动solarGroup/universeGroup的invisiblePhenomena子组visible
